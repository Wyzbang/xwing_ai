
Adding a new ship

1. Create new image
	Format:		PNG
	Size: 		100x100 pixels
2. Add node to generator/ships.xml
3. Run "ant compile"


Distribute:
1. Run "ant dist"
2. File is saved to dist/xwing_ai.zip
