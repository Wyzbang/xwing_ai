
Adding a new ships

1. Create new images in web/img
	Format:		PNG
	Size: 		100x100 pixels
2. Add node to generator/<filename>.xml
3. Run "ant compile"


Distribute:
1. Run "ant dist"
2. File is saved to dist/xwing_ai.zip
