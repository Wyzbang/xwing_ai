Purpose:

Provide a AI control of one or more ships for the X-Wing Miniture Game

Setup:

Setup game per standard rules, choosing any squadrons

Usage:

For each AI controlled ship:
  1. Select the ship type for the AI controlled ship
  2. Pick a target (human controlled) ship for the AI ship to attach
  3. Select the arrow that matches the direction and orientation of target (human) ship
    • direction = closest cardinal direction of the target (human) ship in relation to the AI ship
    • orientation = if the target (human) ship is closing or retreating from the AI ship
  4. Choose Action
  5. Complete round per standard rules
