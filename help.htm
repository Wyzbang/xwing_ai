Purpose:

Provide a AI control of one or more ships for the X-Wing Miniture Game

Setup:

Setup game per standard rules, choosing any squadrons and or missions.

Usage:

For each AI controlled ship:
  1. Select the ship type for the AI controlled ship
  2. Pick a target (human controlled) ship for the AI ship to attach
  3. Select the arrow that matches the direction and orientation of target (human) ship
    • direction = closest cardinal direction of the target (human) ship in relation to the AI ship
    • orientation = if the target (human) ship is closing or retreating from the AI ship
  4. Choose Action
  5. Complete round per standard rules

Picking Target:

There are multiple options for picking targets
  1. Always pick the nearest possible target
  2. Randomize tarhry by rolling a 6-sided die:
    • 1-2: take on the closest one
    • 3-4: tako on the most damaged one
    • 5-6: take on the highest skilled pilot.

Choosing action:

Tie Fighter

If this will put the Tie Figher into or out of firing position choose Barrel-Roll
Else if already in a firing position choose Focus
Else choose Evade

Tie Advanced

Treat Target-Lock as a free action, target same ship as chosen for manuver. Clear Target lock at end of turn.
If this will put the Tie Figher into or out of firing position choose Barrel-Roll
Else if already in a firing position choose Focus
Else choose Evade

X-Wing

Treat Target-Lock as a free action, target same ship as chosen for manuver. Clear Target lock at end of turn.
Always use Focus

Y-Wing

Treat Target-Lock as a free action, target same ship as chosen for manuver. Clear Target lock at end of turn.
Always use Focus

NOTES:

  • Always choose the human control ship(s) manuvers, before chosing AI manuvers
  • AI ships never get stress token, but do not perform any actions after a RED manuver.
  • For secondary weapons or optional pilot skills, always use them when applicable
