Overview
-------

Simple web based for AI control of ships in the "Star Wars x-Wing Minitures game".


------------
Installation
------------

Extract zip file to any location
Open "index.html" in a browser

------------
Instructions
------------

Setup "X-Wing Miniatures" according to standard rules and/or missions.

For each game round:
- Human picks manuvers first
- Click on the arrow that best reprents the location and heading of the Human controlled ship from the AI Ship
- Choosing action for the AI ship
- Attach the nearest Human Ship (or mission target)
- Roll damage and defense dice for both Human and AI ships
